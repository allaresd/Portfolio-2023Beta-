Color Scheme

Secondary Colour: #25d3f4

This colour can be applied to dividing lines.

Primary Colour: white

This is the brand color. Apply it in the main buttons, outlines while hovering, section title, decoration lines and in-text-links.

Secondary Colour: #fb4e03

This colour can be applied to text, such as navigation link and secondary navigation.

Secondary Colour: #979797

You can use this color in less important text, such as some auxiliary explanatory text.

Secondary Colour: #2c2c2c

You can use this colour in important text, such as body text.
